# 介绍

[![Jenkins Plugin](https://img.shields.io/jenkins/plugin/v/dingtalk.svg?label=Version)](https://plugins.jenkins.io/dingtalk)

在 Jenkins 中使用钉钉机器人发送消息。

## 特性

1. 简单配置即可用
2. 支持所有类型的项目
3. 支持多种消息类型
